# 🏁 Математическое приключение

Интерактивный помощник по математике для школьника 2 класса.
Курс построен по учебнику **А. С. Пчёлко, Г. Б. Поляк «Арифметика, 2 класс»**
(материал переработан простым языком, не копия). Темы про **гольф и машины**.

## Как запустить

### Способ 1 — самый простой (двойной клик)
Открой файл `index.html` двойным щелчком — приложение откроется в браузере.
Весь прогресс сохраняется прямо в браузере (localStorage).

### Способ 2 — через локальный сервер (рекомендуется)
1. Открой терминал в этой папке.
2. Выполни:
   ```bash
   python3 server.py
   ```
3. Открой в браузере: <http://localhost:8123>

Никакой установки и интернета не нужно.

## Что внутри
- **Главная** — приветствие, уровень, прогресс, кнопки занятий, монеты/звёзды, фраза дня.
- **Карта курса** — 6 тем-островов.
- **Урок** — объяснение → пример с разбором по шагам → мини-тренировка → проверка → итог → награда.
- **Тренажёр** — задачи 3 уровней (лёгкий/средний/сложный) с подсказками и разбором ошибок.
- **5 минут** — короткая ежедневная тренировка из 7 заданий.
- **Слабые места** — темы, где много ошибок, с предложением их прокачать.
- **Профиль** — прогресс, достижения, рекомендации.
- **Отчёт для взрослых** — простой обзор для родителя/учителя.

## Возможности v2
- **Новые типы заданий:** сопоставление (выбор пар) и задача с пропуском.
  *Примечание:* «перетаскивание» сделано как удобный для планшета **тап-вариант сопоставления** (выпадающие списки), а не HTML5 drag — так надёжнее работает пальцем.
- **Озвучка** объяснений и заданий (кнопки 🔊, синтез речи браузера). Включается/выключается в Профиле.
- **Анимация наград** (конфетти) и **разблокировка островов по очереди** (можно отключить в настройках — «открыть все сразу»).
- **Генератор задач** — бесконечная практика со случайными числами в нужном диапазоне для каждой темы.
- **Адаптивная сложность** — режим «🤖 Авто-подбор» сам выбирает уровень по последним ответам.
- **Несколько профилей детей** + кабинет учителя (обзор всех детей в отчёте для взрослых).
- **Перенос прогресса между устройствами** — экспорт/импорт по коду (Профиль → «Получить/Ввести код»).
- **PWA** — можно «установить как приложение» и работать офлайн (только при запуске через сервер/https, не из `file://`).

> **Про настоящую облачную синхронизацию:** живая синхронизация между устройствами в реальном времени требует сервера/БД (например, Firebase). Здесь сделан офлайн-перенос по коду. Чтобы подключить Firebase позже — точки интеграции `exportCode()` / `importCode()` в `js/app.js` (заменить на запись/чтение в облако).

## Облачная синхронизация через Firebase (необязательно)
Теперь есть **настоящая** синхронизация между устройствами. Нужна минимальная настройка (без npm — SDK грузится с CDN):

1. Создай проект на <https://console.firebase.google.com>.
2. Включи **Authentication → Email/Password**.
3. Создай **Firestore Database**.
4. В настройках проекта добавь Web-приложение (`</>`), скопируй `firebaseConfig`.
5. Вставь значения в файл [`js/firebase-config.js`](js/firebase-config.js) (подробная инструкция внутри файла).
6. Запусти приложение **через сервер** (`python3 server.py`) — облако не работает из `file://`.
7. В Профиле появится раздел «☁️ Облако»: зарегистрируйся по почте/паролю — и прогресс будет на всех устройствах автоматически.

Пока конфиг не заполнен — приложение работает как обычно, а в Профиле остаётся перенос прогресса по коду.

Рекомендуемые правила Firestore (каждый видит только свои данные):
```
rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /users/{uid} {
      allow read, write: if request.auth != null && request.auth.uid == uid;
    }
  }
}
```
